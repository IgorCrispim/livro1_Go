# 📚 Let's Go — Minha Jornada de Estudos

Este repositório documenta minha jornada de aprendizado com o livro **Let's Go: Learn to build professional web applications with Go**, de **Alex Edwards**.

O objetivo é registrar minha evolução enquanto estudo o desenvolvimento de aplicações web utilizando **Go**, implementando cada capítulo, praticando os conceitos apresentados e compartilhando minha evolução.

## 🚀 Sobre o projeto

Durante o livro será desenvolvido o **Snippetbox**, uma aplicação web para criar, armazenar e compartilhar pequenos trechos de texto, inspirada em plataformas como Pastebin e GitHub Gists.

Ao longo do desenvolvimento serão abordados diversos conceitos importantes do ecossistema Go, incluindo:

* Estruturação de projetos
* Servidores HTTP
* Roteamento
* Templates HTML
* Manipulação de formulários
* Banco de dados
* CRUD
* Middleware
* Autenticação de usuários
* Sessões
* HTTPS
* Boas práticas para aplicações web em Go

## 🎯 Objetivo

Mais do que apenas reproduzir o código do livro, este repositório serve para:

* Consolidar o aprendizado de Go;
* Entender como aplicações web são estruturadas;
* Explorar boas práticas utilizadas em projetos reais;
* Criar um histórico da minha evolução.
